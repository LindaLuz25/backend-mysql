# E-Commerce Backend
![Github licence](http://img.shields.io/badge/license-MIT-blue.svg)

## Description 
This project is the creation of the back end for an e-commerce site. This application used Express.js API and Sequelize to interact with a MySQL database.

## Installation 
The user should clone the repository from GitHub. This application requires Node.js, Express.js, and Sequelize. To start application run npm start. 

## Usage 
This application will allow users to view, add, edit, and delete categories, products, and tags.
